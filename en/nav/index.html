Help us add links ​

How to modify the navigation?

docs/en/nav/data.ts is the database for navigation, just follow the JSON format for editing.
If you are a webmaster, please add MCJPG Organization to the appropriate location on your site，MCJPG Organization thanks a lot

TIP: What is JSON ?

Where are the icons placed? How is it referenced?

Icons are located in docs/public/icons/nav/ They are already categorized, just add them by category.

json
    # Here's an example, which can also be edited in the same format
    {
        icon: '/icons/nav/百科/中文 Minecraft Wiki.ico',
        title: '中文 Minecraft Wiki',
        desc: 'Minecraft最著名的、最权威的非官方百科网站',
        link: 'https://zh.minecraft.wiki/',
    },
